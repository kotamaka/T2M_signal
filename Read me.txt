วิธีใช้งาน T2M
วิธีใช้ติดตั้ง
1. เข้าหน้า login ,สมัครสมาชิก ,ยืนยันอีเมล์
2. เมื่ออยู่หน้า profile ให้ download Metatrade5 T2M_client-main 
3. ติดตั้ง Metatrade5 

วิธีใช้งาน
1. แตกไฟล์ T2M_client-main
2. คลิกขวา run as administrator app.exe 
3. เข้าหน้า login
4. ก็อป passcode 
5. ทำการ login ใส่ email , passcode
6. ตั้งค่า lot size
7. *เลือกโหมด*
	swap คือ เปิด buy ปิด sell พอปิดออเดอร์แล้ว จะเปิด sell ที่ราคาเดิมต่อทันที
	one-side คือ จะให้เลือกเปิด buy หรือ sell
	ถ้าเลือก buy ระบบจะเข้า buy ออก sell
	ถ้าเลือก sell ระบบจะเข้า sell ออก buy
8. เมื่อโปรแกรมพร้อมใช้งาน copy url ไปวางที่ webhook
9. copy json ไปวางที่ ข้อความ trading view
10. enjoy

วิธีปิดโปรแกรม
1.กด ctrl+c แล้วกด y
**หากไม่ทำการกด session จะค้างในระบบจะไม่สามารถล็อกอินครั้งต่อไปได้**

วิธีเติมวัน
1. เข้าไปที่หน้า profile และกดเลือก plan
2. สแกน qr code 
3. รับวันใช้งาน

วิธีขอ password ใหม่
1. ไปที่หน้า login กดลืมรหัสผ่าน
2. ใส่อีเมล์ที่เคยสมัคร
3. เข้าไปที่เมล์แล้วกดลิ้งค์
4. ระบบจะส่งมาหน้าใส่รหัสผ่านใหม่
5. เสร็จสิ้น
